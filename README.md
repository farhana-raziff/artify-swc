# 🎨 Artify: Workshop Reservation System

Artify is a JavaFX-based application that allows users to browse art workshops, make bookings, and manage their reservation history.  
Admins can add, edit, and monitor workshop details efficiently.

## 📖 Description
This system replaces manual booking methods with a centralized digital platform, improving accuracy and user experience.

## 🚀 Features
- User registration and login
- Workshop listing with schedule and price
- Booking page with session and add-ons
- Booking history with search and cancellation
- Admin panel for workshop management

## 🛠️ Tech Stack
- JavaFX
- MySQL
- DAO pattern for database operations
